Scoped Styles

¿Qué son los estilos Scoped?

Los estilos "scoped" están encapsulados dentro de un componente, lo que significa que no afectan a otros componentes del proyecto. Esta técnica es ampliamente usada en frameworks como Angular, React y Vue.

Ventajas principales

  • Evita conflictos globales de estilo.
  • Fácil mantenimiento de componentes.
  • Reutilización sencilla.
  • Encapsulamiento real del diseño.

Ejemplos en frameworks

<template>
  <div class="card">Hola desde Vue</div>
</template>

<style scoped>
.card {
  background-color: lightyellow;
  padding: 1rem;
  border-radius: 8px;
}
</style>

// Card.module.css
.card {
  background-color: lightblue;
  padding: 1rem;
  border-radius: 8px;
}

// Card.jsx
import styles from './Card.module.css';
export default function Card() {
  return <div className={styles.card}>Hola desde React</div>;
}


// card.component.html
<div class="card">Hola desde Angular</div>

// card.component.css
.card {
  background-color: lightgreen;
  padding: 1rem;
  border-radius: 8px;
}

Conclusión

Scoped Styles permiten una arquitectura limpia y escalable en proyectos frontend modernos. Combinados con metodologías como ITCSS y CSS modular, mejoran la mantenibilidad del código.
