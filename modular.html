CSS Modular

¿Qué es?

CSS Modular implica separar estilos en archivos y componentes reutilizables, minimizando colisiones y facilitando mantenimiento.

¿Cómo funciona?

Cada componente tiene su archivo de estilos: `button.module.css`, `card.module.css`, etc.
