CSS Modular

¿Qué es CSS Modular?

CSS Modular es una forma de organizar nuestros estilos CSS para que sean más fáciles de mantener y reutilizar. Es como tener una caja de herramientas donde cada herramienta tiene su propio lugar.

¿Por qué es importante?

  • Evita que los estilos se mezclen entre sí
  • Facilita el trabajo en equipo
  • Hace más fácil encontrar y arreglar errores
  • Permite reutilizar código

Ejemplo Básico

Imagina que tenemos un botón. En lugar de escribir todos los estilos en un solo archivo, podemos crear un archivo específico para botones:

/* button.css */
.boton {
  color: white;
  background: blue;
  padding: 10px;
}

Ventajas para Estudiantes

Organización

Es más fácil encontrar tus estilos cuando están organizados por componentes.

Aprendizaje

Te ayuda a entender mejor cómo funcionan los componentes en CSS.

Consejos para Empezar

  1. Empieza con componentes pequeños (botones, tarjetas)
  2. Usa nombres descriptivos para tus clases
  3. Mantén tus archivos CSS organizados por tipo de componente
  4. No tengas miedo de experimentar y cometer errores

Ejemplo Práctico

Aquí hay un ejemplo simple de cómo podrías estructurar tus archivos CSS:

css/
  ├── botones.css
  ├── tarjetas.css
  ├── navegacion.css
  └── estilos.css

Comparación: Antes y Después

Antes (CSS Normal) 
/* Un solo archivo grande */
.boton { ... }
.tarjeta { ... }
.navegacion { ... }
.footer { ... }
/* ¡Todo mezclado! */
Después (CSS Modular) 
/* Archivos separados */
/* botones.css */
.boton { ... }

/* tarjetas.css */
.tarjeta { ... }

/* navegacion.css */
.navegacion { ... }

Ejercicio Práctico

Intenta crear estos componentes modulares:

Botón
.mi-boton {
  padding: 10px;
  border-radius: 5px;
}
Tarjeta
.mi-tarjeta {
  border: 1px solid #ccc;
  padding: 15px;
}
Navegación
.mi-nav {
  display: flex;
  list-style: none;
}
